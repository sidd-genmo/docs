# Genmo Video Generation API

Welcome to Genmo's API documentation. Our API enables you to generate high-quality videos from text descriptions using state-of-the-art AI models.

## Key Features

- Text-to-video generation using advanced AI models
- Two specialized models: Standard and Mochi
- Real-time status updates via webhooks
- Simple REST API interface

## Getting Started

To start using the Genmo API, you'll need:

1. An API key for authentication
2. A text prompt describing your desired video
3. A webhook URL (optional) to receive status updates

## Example Usage

Here's a simple example of generating a video:

```bash
curl -X POST https://api.genmo.ai/create \
  -H "Authorization: YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "A serene lake at sunset with mountains in the background",
    "model": "standard"
  }'
```
